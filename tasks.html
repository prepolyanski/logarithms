Задача 1 · Уравнение · ЕГЭ 2023
Решите уравнение:
$$4^x - 5 \cdot 2^x + 4 = 0$$
📌 Метод замены переменной
Решение
1
Запишем $4^x = (2^x)^2$. Введём замену: $t = 2^x,\; t > 0$
2
$t^2 - 5t + 4 = 0$
3
$(t-1)(t-4) = 0 \;\Rightarrow\; t = 1$ или $t = 4$
4
$2^x = 1 = 2^0 \Rightarrow x = 0$;   $2^x = 4 = 2^2 \Rightarrow x = 2$
ОТВЕТ
$x = 0,\quad x = 2$

Задача 2 · Показательное неравенство
Решите неравенство:
$$\left(\frac{1}{3}\right)^{2x-1} \leq 9^{x+2}$$
📌 Привести к одному основанию
Решение
1
$\left(\frac{1}{3}\right)^{2x-1} = 3^{-(2x-1)}$,   $9^{x+2} = 3^{2(x+2)}$
2
$3^{-(2x-1)} \leq 3^{2x+4}$;   основание $3 > 1$, знак сохраняется:
3
$-(2x-1) \leq 2x+4 \;\Rightarrow\; -2x+1 \leq 2x+4$
4
$-3 \leq 4x \;\Rightarrow\; x \geq -\dfrac{3}{4}$
ОТВЕТ
$x \in \left[-\dfrac{3}{4};\; +\infty\right)$

Задача 3 · Уравнение · ЕГЭ 2022
Решите уравнение:
$$\log_2(x+3) + \log_2(x-1) = 5$$
📌 Сложение логарифмов + ОДЗ
Решение
1
ОДЗ: $x+3>0$ и $x-1>0 \;\Rightarrow\; x > 1$
2
$\log_2[(x+3)(x-1)] = 5$
3
$(x+3)(x-1) = 2^5 = 32$
4
$x^2+2x-3 = 32 \;\Rightarrow\; x^2+2x-35=0$
5
$x=5$ или $x=-7$ → $x=-7 \notin$ ОДЗ ✗
ОТВЕТ
$x = 5$
✓

Решение

2
$t^2 - 10t + 9 \leq 0$
3
$(t-1)(t-9) \leq 0 \;\Rightarrow\; 1 \leq t \leq 9$
4
$3^0 \leq 3^x \leq 3^2$, основание $3>1$
5
$0 \leq x \leq 2$
ОТВЕТ

Задача 6 · Логарифмич. неравенство
Решите неравенство:
$$\log_2 x + \log_2(x - 3) \leq 2$$
📌 ОДЗ + сложение логарифмов
Решение
1
ОДЗ: $x>0$ и $x-3>0 \;\Rightarrow\; x > 3$
2
$\log_2[x(x-3)] \leq 2 = \log_2 4$
3
Основание $2>1$: $x(x-3) \leq 4$
4
$x^2-3x-4 \leq 0 \;\Rightarrow\; (x-4)(x+1) \leq 0$
5
$-1 \leq x \leq 4$; пересечение с ОДЗ $(x>3)$: $3 < x \leq 4$
ОТВЕТ
$x \in (3;\; 4]$

Задача 7 · Уравнение · ЕГЭ 2024
Решите уравнение:
$$2^{x+1} + 2^{3-x} = 10$$
📌 Замена + квадратное уравнение
Решение
1
$2 \cdot 2^x + \dfrac{8}{2^x} = 10$; замена $t = 2^x > 0$
2
$2t + \dfrac{8}{t} = 10 \;\Rightarrow\; 2t^2 - 10t + 8 = 0$
3
$t^2 - 5t + 4 = 0 \;\Rightarrow\; (t-1)(t-4)=0$
4
$t=1: \;2^x=1=2^0 \Rightarrow x=0$ ✅
5
$t=4: \;2^x=4=2^2 \Rightarrow x=2$ ✅
ОТВЕТ
$x = 0,\quad x = 2$

Задача 8 · Лог. неравенство · ЕГЭ 2023
Решите неравенство:
$$\log_{0{,}5}(x+1) \geq -3$$
📌 Основание меньше 1 — знак меняется
Решение
1
ОДЗ: $x + 1 > 0 \;\Rightarrow\; x > -1$
2
$0{,}5 = \tfrac{1}{2} \in (0;1)$ → при потенцировании знак меняется
3
$x+1 \leq \left(\tfrac{1}{2}\right)^{-3} = 2^3 = 8$
4
$x \leq 7$; пересечение с ОДЗ: $-1 < x \leq 7$
ОТВЕТ
$x \in (-1;\; 7]$

Задача 10 · Лог. неравенство · ЕГЭ 2024
Решите неравенство:
$$\log_2 4x - \log_4 x \geq 3$$
📌 Разные основания → формула перехода
Решение
1
ОДЗ: $x > 0$
2
$\log_4 x = \dfrac{\log_2 x}{\log_2 4} = \dfrac{\log_2 x}{2}$
3
$\log_2 4x = \log_2 4 + \log_2 x = 2 + \log_2 x$
4
Замена $t = \log_2 x$: $\;(2+t) - \dfrac{t}{2} \geq 3$
5
$2 + \dfrac{t}{2} \geq 3 \;\Rightarrow\; \dfrac{t}{2} \geq 1 \;\Rightarrow\; t \geq 2$
6
$\log_2 x \geq 2 \;\Rightarrow\; x \geq 4$
ОТВЕТ
$x \in [4;\; +\infty)$
